# 🎓 Student Registration System

A Full Stack Student Registration System developed using HTML, CSS, JavaScript, Node.js, Express.js and MySQL. This application allows administrators to register, manage and update student records through a secure dashboard.

## 🚀 Features

- Student Registration
- Secure Admin Login
- Dashboard
- View Student Details
- Update Student Details
- Delete Student Records
- MySQL Database Integration
- Responsive User Interface

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- MySQL

## 📂 Project Structure

```
controllers/

README.md
```

```bash
npm install
npm start
```
